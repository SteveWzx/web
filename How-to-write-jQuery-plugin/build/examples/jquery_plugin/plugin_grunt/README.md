## Getting Started
Download the [production version][min] or the [development version][max].

[max]: https://raw.github.com/i5ting/How-to-write-jQuery-plugin/master/dist/i5ting-mobile.js

```html

<script>
jQuery(function($) {
  $.awesome(); // "awesome"
});
</script>
```
