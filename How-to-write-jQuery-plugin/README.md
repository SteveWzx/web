# How-to-write-jQuery-plugin ?


如何编写jQuery插件，以及代码重构过程


## 目录

- src      源文件
- examples 示例
- build    指src里md文件build成pdf和epub之类格式的输出物的目录

## 教程

- src/jquery.plugin.md



## 编译


	bundle install
	rake build


then open build/jquery.plugin.html
