使用前的准备工作
第一步 在你的页面head里引入相关css, 如下所示
<link type="text/css" rel="stylesheet" href="main.css" />
第二步 在你的页面里, 通常是在页面底部引入jquery和ant-widget.js, 如下所示
<script type="text/javascript"src="http://static.wkimg.com/common/lib/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript" src="http://static.wkimg.com/common/lib/widget/2.0.1/ant-widget.js"></script>
第三步 在你的页面里就可以用相关插件函数了, 如下所示
        
        <script>
            var flag = $.isIE6();
            if(flag){
                //some code;
            }
            
            $('.slide-box').slide();
            ...
        </script>

操作cookie
$.cookie(cookie名字, cookie值, options)
参数 数据类型 默认值 参数说明
cookie名字(必须) string 无

要获取或者要设置的cookie的名
cookie值(必须) string 无

设置cookie的值
expires(可选) number 1天

cookie过期时间
domain(可选) string 当前域

设置cookie所在域
path(可选) string 根目录

设置cookie所在的网站目录
返回值
            获取cookie的值
用法
            //设置cookie
            
            $.cookie('cookieName', 'zhangsan');
            
            //设置带有可选参数的cookie
            
            $.cookie('cookieName', 'zhangsan', {expires: 2, domain: '/'});
            
            //读取cookie
            
            $.cookie('cookieName');
            
            //删除cookie
            
            $.cookie('cookieName', {expires: -1});
选项卡
$(jquery选择器).tab(options)
参数 数据类型 默认值 参数说明
jquery选择器(必须) string 无

标准的jquery选择器
type(可选) string mouseover

鼠标事件名称
triggerClass(可选) string current

选项卡切换时，当前选项的class名字
tab(可选) string li

指定切换项的元素,标准jquery选择器, 如.tab > li
panelClass(可选) string panel

指定对应切换项目元素的class名字
defaultPanel(可选) number 0(默认显示第一个)

显示指定索引值的面板, 索引值从0开始
slideTab(可选) object trigger: .factor
sliding: false
vertical: false
speed: 300ms

此项是复合参数。开启tab项的滑动效果, 包含4个参数
trigger  具有滑动效果的元素class
sliding  是否打开tab项的滑动效果
vertical  是否垂直方向滑动
speed  滑动速度

opacity(可选) boolean false

切换时是否带透明度效果
delay(可选) number 200ms

当事件是mouseover时,鼠标经过时的延时时间
operate(可选) function function(){}

非异步方式下的回调函数.该函数有一个参数, 是当前项的索引值
asyn(可选) object state: false
url: a.php
type: get
dataType:json
data: [ ]
error: function(){}
success: function(){}

此项是复合参数, 选项卡是否开启ajax调用
state  是否开启ajax调用, 默认false, 不开启
url  请求的后台url.多个url以逗号分开, 和tab项一一对应, 当某项无ajax请求时, 为空值.如'" ","a.php","b.php"'代表第一个tab项无ajax请求
type  请求后台的方式, get | post
dataType  后台返回的数据格式, json | jsonp | text
data  附加到url后的参数, 数组格式, 和tab项一一对应, 当某项无参数时, 值为0. 如[0,'a=2','a=3']代表第一个tab项无参数.多个参数以&符分开, 如['a=1&b=2']
error  请求失败时要执行的函数
success  请求成功时要执行的函数, 此函数返回2个参数:第一个是当前的数据面板元素, jquery对象;第二个是后台返回的数据

返回值
            无返回值
用法
            //默认调用
            
            $('.tab-box').tab();
            
            //带有可选参数的调用
            
            $('.tab-box').tab({
            
                type:'mouseover',
                panelClass: 'panel', 
                delay: 150
                
            });
            
            //tab项带滑动效果的调用
            
            $('.tab-box').tab({
            
                type:'mouseover',
                panelClass: 'panel', 
                slideTab: {sliding: true, trigger: '.on', speed: 300}
                
            });
            
            //开启ajax效果的调用
            
            $('.tab-box').tab({
            
                type:'mouseover',
                panelClass: 'panel',
                asyn: {state: true, url: 'a.php', dataType: 'jsonp', success: function(obj, data){
                      //obj.html(data);
                 }
               }
                
            });

参数 数据类型 默认值 参数说明

焦点图是否具有动画效果

是否可以垂直方向运动。此项当animate为true时, 有效

是否具有透明度变化效果, 当此项打开时, animate自动关闭。

rightButton(可选) string .right_button

trigger(可选) boolean false

triggerClass(可选) string current

手动切换小图标的class名字
dataAttr(可选) string title

存放图片标题的自定义属性
dataPanel(可选) string .text

type(可选) string mouseover

auto(可选) boolean true

speed(可选) number 300ms

time(可选) number 3000ms

返回值
            无返回值
用法
            //默认用法
            
            $('.slide').slide({loop: true});
            
            //具备横向动画效果的用法
            
            $('.slide').slide({
            
                type: 'mouseover', 
                triggerClass: 'current',
                loop: true,
                animate: true,
                time: 3000
                
            })
			
            //具备竖直方向动画效果的用法
            
            $('.slide').slide({
            
                type: 'mouseover', 
                triggerClass: 'current', 
                loop: true,
                animate: true,
                vertical: true,
                time: 3000
                
            })
			
            //具备透明度变化效果的用法
            
            $('.slide').slide({
            
                type: 'mouseover', 
                triggerClass: 'current', 
                loop: true,
                animate: false,
                opacity: true,
                time: 3000
                
            })

图集相册效果
$(jquery选择器).photoAlbum(options)//此插件采用图片预先加载方式，保证图片查看顺畅
参数 数据类型 默认值 参数说明
jquery选择器(必须) string 无

标准jquery选择器
photoPanel(必须) string .photo-origin

存放大图的容器, 合法的jquery选择器, 如.photo-origin
dataPanel(可选) string .photo-info

存放大图描述的容器, 合法的jquery选择器, 如.photo-info
preButton(可选) array ['.curPrevBtn', '.prevBtn']

向前查看图片的按钮class或者id, 合法的jquery选择器, 这里以数组方式传入
nextButton(可选) array ['.curNextBtn', '.nextBtn']

向后查看图片的按钮class或者id, 合法的jquery选择器, 这里以数组方式传入
animate(可选) object opacity: false
speed: 400

此项是复合参数, 设置查看图片时是否带动画效果(默认是透明度变化), 包含2个属性.
opaicty: 是否打开透明度, 默认false
speed: 透明度变化速度(毫秒数), 默认400毫秒
vertical(可选) boolean false

缩略图是否在垂直方向展示
holdPage(可选) boolean false

刷新页面后, 是否停留在上次查看的图片上。此项通过hash记住当前状态
mouseWheel(可选) boolean false

是否开启鼠标滚轮来查看图片
dataAttr(必须) json {"url": "", "data" : ""}

这里以标准json方式存放数据, url存放每张大图的真实地址, data存放每张大图的描述信息
numPanel(可选) string .number

存放每张图片序号的容器, 合法jquery选择器, 如.number
totalPanel(可选) string .total

存放图片总数的容器, 合法jquery选择器, 如.total
triggerClass(可选) string current

处于当前状态下缩略图的class名字
pages(可选) object pagePanel:'.photo-page'
numTag:'a'
triggerClass:'current'

此项是复合参数, 设置图集是否展示全部页码。目的是增强查看图集的快捷性。包含三个参数: pagePanel: 存放页码的容器, 合法的jquery选择器.
numTag: 输出每个页码所使用的标签, 默认a标签.
triggerClass: 当前页码的class名字。
operate(可选) function 无

图片浏览时所执行的回调函数, 此函数有2个参数,每一张图片的索引值和图片总数
返回值
            无返回值
用法
            //默认用法
            
            $('.photo').photoAlbum();
            
            //带有可选参数的用法
            
            $('.photo').photoAlbum({
            
                preButton: ['.thumb-prev-btn', '.photo-prev'],
                nextButton: ['.thumb-next-btn', '.photo-next'],
                pages: {pagePanel: '.page-num', triggerClass: 'current'},
                animate: {opacity: true, speed: 800},
                operate: function(index, length){
                    if(index === length - 1){
                        //some code
                    }
                    if(index === 0){
                        //some code
                    }
                }
                
            });

$('input').checkValue(options)
参数 数据类型 默认值 参数说明
defaultValue(可选) Array [ ]

此参数可以设置文本框的默认值, 数组形式, 如['请输入用户名', '请输入年龄']
focusCss(可选) object { }

此参数是复合类型.可以对获取焦点的文本框进行样式设置.写法完全遵循css对象表示法.如:
focusCss: {color: '#666'}
blurCss(可选) object { }

此参数是复合类型.可以对失去焦点的文本框进行样式设置.写法完全遵循css对象表示法.如:
blurCss: {color: '#999'}
返回值
            无返回值
用法
            //默认用法
            $('input.keyword').checkValue();
            
            //带可选参数的用法
            $('input.keyword').checkValue({
                defaultValue: ['请输入用户名'],
                focusCss: {
                    borderColor: '#ff6600',
                    color: '#333',
                    width: function(){
                        $(this).animate({width: 300}, 200);
                    }
