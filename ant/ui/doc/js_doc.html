版本说明

2.0.1版本的更新说明：

1. 去掉了loopSlide。

2. 改写了slide，把loopSlide的部分效果融合进了slide，并且性能更好。

3. 凡是带有leftButton和rightButton参数的，都改为了prevButton和nextButton

4. 优化了monitor。

导读

ant widget是基于jquery的插件库，使用简单，通过配置相关参数，配合ant css，能快速开发出界面效果，可以做到一种结构，多种效果。


部分例子
$('.slide-box').slide({
    animate: true
    time: 4000
})

...
准备工作
第一步 在你的页面head里引入插件所需的css, 比如：

引入css
<link rel="stylesheet" type="text/css" href="http://static.wkimg.com/common/ui/base/ant-base.css" />
第二步 按照插件要求写好html结构，比如：

html结构
<div class="ant-slide">
  <div class="ant-slide-pic-wrap">
    <ul class="ant-slide-pic">
      <li><img src="xxx/1.jpg" width="600" height="200" /></li>
    </ul>
  </div>
  <ul class="ant-slide-trigger">
      <li class="current"></li>
  </ul>
</div>
第三步 在你的页面, 通常是在底部引入指定的jquery和ant-widget文件, 如下所示

引入js
<script type="text/javascript" src="http://static.wkimg.com/common/lib/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript" src="http://static.wkimg.com/common/lib/widget/2.0.1/ant-widget.js"></script>
第四步 调用插件相关方法即可，如下所示

一个简单示例
<script type="text/javascript">

    $('.ant-slide').slide({
       animate: true
   })
   
</script>

选项卡$('.ant-tab').tab(options)
options参数列表
参数 数据类型 默认值 参数说明
type string mouseover

鼠标事件名称
triggerClass string current

选项卡切换时，当前选项的class名字
tab string li

指定切换项的元素,标准jquery选择器, 如.tab > li
panelClass string panel

指定对应切换项目元素的class名字
defaultPanel number 0(默认显示第一个)

显示指定索引值的面板, 索引值从0开始
slideTab object trigger: .factor
sliding: false
vertical: false
speed: 300ms

此项是复合参数。开启tab项的滑动效果, 包含4个参数
trigger  具有滑动效果的元素class
sliding  是否打开tab项的滑动效果
vertical  是否垂直方向滑动
speed  滑动速度

opacity boolean false

切换时是否带透明度效果
delay number 200ms

当事件是mouseover时,鼠标经过时的延时时间
operate function function(){}

非异步方式下的回调函数.该函数有一个参数, 是当前项的索引值
async object state: false
url: a.php
type: get
dataType:json
data: [ ]
error:function
success:function

此项是复合参数, 选项卡是否开启ajax调用
state  是否开启ajax调用, 默认false, 不开启
url  请求的后台url.多个url以逗号分开, 和tab项一一对应, 当某项无ajax请求时, 为空值.如'" ","a.php","b.php"'代表第一个tab项无ajax请求
type  请求后台的方式, get | post
dataType  后台返回的数据格式, json | jsonp | text
data  附加到url后的参数, 数组格式, 和tab项一一对应, 当某项无参数时, 值为0. 如[0,'a=2','a=3']代表第一个tab项无参数.多个参数以&符分开, 如['a=1&b=2']
error  请求失败时要执行的函数
success  请求成功时要执行的函数, 此函数返回2个参数:第一个是当前的数据面板元素, jquery对象;第二个是后台返回的数据

返回值
无返回值
用法 

html结构
<div class="tab-box">
  <div class="ant-tab">
     <ul class="ant-yahei ant-clearfix">
        <li class="current">皮肤科</li>
        <li>神经内科</li>
        <li>神经外科</li>
        <li>血液科</li>
     </ul>
  </div>
  <div class="panel">panel1</div>
  <div class="panel" style="display:none;">panel2</div>
  <div class="panel" style="display:none;">panel3</div>
  <div class="panel" style="display:none;">panel4</div>
</div>

调用方法
$('.tab-box').tab({
    type:'mouseover',
    panelClass: 'panel', 
});

图集相册$('.ant-photos').photoAlbum(options)
options参数列表
参数 数据类型 默认值 参数说明
photoPanel(必须) string .photo-origin

存放大图的容器, 合法的jquery选择器, 如.photo-origin
dataPanel string .photo-info

存放大图描述的容器, 合法的jquery选择器, 如.photo-info
prevButton array ['.curPrevBtn','.prevBtn']

向前查看图片的按钮class或者id, 合法的jquery选择器, 这里以数组方式传入
nextButton array ['.curNextBtn','.nextBtn']

向后查看图片的按钮class或者id, 合法的jquery选择器, 这里以数组方式传入
animate object opacity: false
speed: 400

此项是复合参数, 设置查看图片时是否带动画效果(默认是透明度变化), 包含2个属性.
opaicty: 是否打开透明度, 默认false
speed: 透明度变化速度(毫秒数), 默认400毫秒
vertical boolean false

缩略图是否在垂直方向展示
holdPage boolean false

刷新页面后, 是否停留在上次查看的图片上。此项通过hash记住当前状态
mouseWheel boolean false

是否开启鼠标滚轮来查看图片
dataAttr(必须) json {"url": "", "data" : ""}

这里以标准json方式存放数据, url存放每张大图的真实地址, data存放每张大图的描述信息
numPanel string .number

存放每张图片序号的容器, 合法jquery选择器, 如.number
totalPanel string .total

存放图片总数的容器, 合法jquery选择器, 如.total
triggerClass string current

numTag:'a'
triggerClass:'current'

此项是复合参数, 设置图集是否展示全部页码。目的是增强查看图集的快捷性。包含三个参数: pagePanel: 存放页码的容器, 合法的jquery选择器.
numTag: 输出每个页码所使用的标签, 默认a标签.
triggerClass: 当前页码的class名字。
operate function 无

图片浏览时所执行的回调函数, 此函数有2个参数,每一张图片的索引值和图片总数

自定义弹出层$('ant-dialog').dialog(options)
options参数列表
参数 数据类型 默认值 参数说明
container string .dialog-bd

对话框内容容器的class或者id, jquery选择器
content string hello world !

对话框内容
close string .dialog-close

对话框关闭按钮的class或者id, jquery选择器
confirm string .dialog-confirm

对话框确认按钮的class或者id, jquery选择器
cancel string .dialog-cancel

对话框取消按钮的class或者id, jquery选择器
operate function 无

点击确定按钮触发的回调函数, 此函数有一个参数, 即遮罩层的id
animate boolean false

是否支持动画
mask boolean true

是否带有遮罩层
opacity number 0.1

设置遮罩层的透明度
用法 

html结构
<div class="ant-dialog">
    <div class="ant-dialog-hd"><a href="javascript:;" class="ant-dialog-close">关闭</a></div>
    <div class="ant-dialog-bd"></div>
    <div class="ant-dialog-extra">
        <a href="javascript:;" class="ant-dialog-cancel">取消</a>
        <a href="javascript:;" class="ant-dialog-confirm">确定</a>
    </div>
</div>

调用方法
$('.ant-dialog').dialog({
   title: '对话框',
   container: '.ant-dialog-bd',
   content: 'hello world!',
   close: '.ant-dialog-close',
   cancel: '.ant-dialog-cancel',
   confirm: '.ant-dialog-confirm',
   animate: false
});
