导读

在ant css里，都以ant-开头。

ant css兼容ie6+及其它现代浏览器。ant css部分采用了css3构建，在兼容性上，采用了“渐进增强”方式处理。

ant css可在现有基础上进行扩展。


部分例子
.ant-yahei {font-family:\5FAE\8F6F\96C5\9ED1;} //微软雅黑
 
.ant-s14 {font-size:14px;} //字体大小14px
 
.ant-ml6 {margin-left:6px;} //margin-left 6px
 
.ant-mr6 {margin-right:6px;} //margin-right 6px;
 
 ...
准备工作
第一步 在你的页面head里引入指定的css, 如下所示

引入css
<link rel="stylesheet" type="text/css" href="http://static.wkimg.com/common/ui/base/ant-base.css" />
第二步 在你的页面, 通常是在底部引入指定的jquery和ant-widget文件, 如下所示

引入js
<script type="text/javascript" src="http://static.wkimg.com/common/lib/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript" src="http://static.wkimg.com/common/lib/widget/2.0.1/ant-widget.js"></script>
第三步 在你的页面里就可以随心写代码了, 如下所示

一个简单示例
<!DOCTYPE html>
<html>
<head>
<meta charset="gbk" />
<title>hello world!</title>
<link type="text/css" rel="stylesheet" href="http://static.wkimg.com/common/ui/base/ant-base.css" />
</head>
<body>

<a class="ant-btn" href="">hello world!</a>

<script type="text/javascript" src="http://static.wkimg.com/common/lib/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript" src="http://static.wkimg.com/common/lib/widget/2.0.1/ant-widget.js"></script>
</body>
</html>
扩展规则
当页面组件样式部分有变化时，采取自定义class方式去扩展，不要重写class名字去覆盖。

一个扩展示例
比如某个页面的一个模块的内容区背景颜色和默认的不一样。那么你可以作如下的扩展。

默认的
<div class="ant-common-item">
   <div class="ant-common-item-hd">模块标题</div>
   <div class="ant-common-item-bd">模块内容</div>
</div>

扩展后
错误做法
.ant-common-item-bd {
   background-color:#999;
}

<div class="ant-common-item">
   <div class="ant-common-item-hd">模块标题</div>
   <div class="ant-common-item-bd">模块内容</div>
</div>

正确做法
.item-bg {
   background-color:#999;
}

<div class="ant-common-item">
   <div class="ant-common-item-hd">模块标题</div>
   <div class="ant-common-item-bd item-bg">模块内容</div>
</div>

我们通过自定义一个item-bg类来修改其背景颜色。这样就可以达到预期效果。不要试图重写原class来实现。

查看UI按钮ant-btn-*

按钮说明
目前按钮有四种高度尺寸。每个尺寸对应四种颜色。高度分别是28px、30px、32px、34px。颜色分别是blue、orange、green、gray。

每个按钮是在基础按钮ant-btn的基础上演变而成的。如：

<a href="###" class="ant-btn ant-btn-middle ant-btn-main-orange"></a> 一个高度为30px的橙色按钮

ant-btn 默认高度28px

ant-btn-middle 高度30px

ant-btn-xmiddle 高度32px

ant-btn-large 高度34px
