什么是规范？

规范是一种约定，是一种态度。在前端开发中，尤其是在有团队参入合作的项目中，规范显的尤为重要。好的规范不仅可以提高工作效率，而且对未来代码的可读性，可扩展性，可维护性具有重大意义。

css命名规范
一、class名以英文或英文缩写开始
二、公用class名以"泛英文"形式命名
三、模块区域统一用"单词-作用域"方式命名

英文或英文缩写开头，"-"分割
.ant-search-item { }
.ant-word-item { }
...
公用class尽量采用"泛英文"命名方式。如下图所示：
上面所示的两幅图，在结构上是相同的，所以可以采用"泛英文"命名方式。

这种命名可能会带来这样的结果，当另一模块和此处相同时，可能又会再起一个class名字。如.founder{ }。这样，可能在样式表中会这样写，.reader, .founder{ }。所以容易造成冗余。
当页面某个部分是无重复的，就可以用特指英文命名。如：友情链接，联系我们等等。
为了统一命名规范，今后大家统一用"单词-作用域"的形式。

模块区域命名示例

    <div class="ant-article-bd">
	<div class="ant-article-item-bd">摘要之类的</div>
        <div class="ant-article-content">内容</div>
     </div>
</div>
css书写规范
一、多用class，少用id
二、css选择符深度最好控制在二级以内
三、影响自身几何属性的属性写在前面，其它属性写在后面

选择符示例
.widget-bd{ }           好

.widget-bd ul{ }      	好

.widget-bd ul li{ }   	不好

.widget-bd li{ }      	好

div.widget-bd span{ } 	不好

.widget-bd span{ }    	好

div#widget ul li{ }   	非常不好
...



写法示例
.widget-box{width:200px;height:100px;border:1px solid #ddd;color:#fff;background:#000;}

.widget-wrap{position:absolute;width:1000px;margin:0 auto;}

width，height，定位，border，margin，padding等这样影响元素自身几何属性的，都写在前面。其它，如color，background，font-size等修饰性属性都写在后面。
js变量命名规范
为了保证项目的进度和开发效率，我们规定统一使用jquery库。虽在开发中我们使用了库，但是对于js代码书写也有很多的要求。

变量统一使用英文命名或英文简写命名，采用驼峰式写法。不许使用匈牙利命名法
var height = 100;		正确

var boxWidth = 10;		正确

var submitBtn = '.submit';	正确

var iBoxWidth = 10;	不正确(匈牙利命名)
...
js代码书写规范
一、采用单一var定义变量。
二、变量与操作符之间，变量与语句之间用空格字符分开。
三、单语句if要加{ }，else要跟在if语句第二个大括号后面。
四、for循环语句中的自增变量提前定义。
五、使用立即执行的匿名函数，防止全局变量污染。
六、dom节点，offsetWidth、offsetHeight、scrollWidth、scrollHeight、clientWidth、clientHeight都用变量来保存。
七、操作innerHTML时，用空字符串临时保存。
八、动态插入dom节点时，用文档片段(documentfragment)来临时保存。
九、对外接口采用模块模式。

采用单一var定义变量。
var i,
    j,
    n,
    key = 0;

变量与操作符之间，变量与语句之间用空格字符分开。
定义变量

var max = 10,
    set = false;
	
书写语句

if(i == 10){
   
   //code
   
}

for(i = 0; i < length; i++){

   //code

}

单语句if要加{ }，else要跟在if语句第二个大括号后面。
if(i == 10){

   return false;
   
}

if(i == 10){

   return false;
   
}else{

   return true;
   
}

for循环语句中的自增变量提前定义。
var i = 0;

for(; i < length; i++){
	
     //code
	
}

使用立即执行的匿名函数，防止全局变量污染
(function(){
   var k = 0,
       top = 10,
       dom = document.getElementById('top');
	
    dom.addEventListener('click', function(){
	
	    return top;
		
    }, false);
	
})()

dom节点，offsetWidth、offsetHeight、scrollWidth、scrollHeight、clientWidth、clientHeight都用变量来保存
var items = document.getElementsByTagName('li'),
    winHeight = window.clientHeight,
    docHeight = document.documentElement.scrollHeight;

操作innerHTML时，用空字符串临时保存。
var str = ''
    node = document.getElementById('list'),
    i = 0,
    length = 10;
	
for(; i < length; i++){

    str += i;
	
}
node.innerHTML = str;

动态插入dom节点时，用文档片段(documentfragment)来临时保存。
var node = document.getElementById('list'),
    fragment = document.createDocumentfragment(),
    nodeItem = null,
    i = 0,
    length = 10;
	
for(; i < length; i++){

    nodeItem = document.createElement('li');
    fragment.appendChild(nodeItem);
	
}
node.appendChild(fragment);


js模块模式，定义对外接口
function module(){
	
    var k = 0,
        init = null;
    
	init = function(){
	    
		return k;
		
	};
	
	return {
        
		init: function(){
		
			init();
			
		}
	}
}

var mod = module();

mod.init();
