一个块级元素默认占据父元素的宽度，width默认为100%，height为auto（当然该块级元素若没有任何内容height经过计算就是0）
外部div的宽度为300px

高度没有设置的话：由line-height（内部为文本情况下）决定 或 正好包含子元素的高度

第二种情况：子元素的高度的累加 + margin-top(除去第一个元素) + margin-bottom(除去最后一个元素)

我margin-top为10px但没有多撑开父元素10px的高度，只是顶开了我父元素的上一个兄弟元素
我上下margin为10px，导致父元素的高度多了20px
我margin-bottom为10px，但没有多撑开父元素10px的高度，只是顶开了我父元素的下一个兄弟元素
兄弟元素(其实这是一个div，我设置成了inline-block，让他包裹住文本，所以并不占body的100%)
我没有给这个div设置高度，只用line-height设置了行高为100px，那么此时他的height就是100px*行数（containing box height）
