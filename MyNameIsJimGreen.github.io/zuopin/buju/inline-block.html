inline-block空白的原因是：在inline-block标签出现的换行、空格等都会解析成一个宽度固定的空白，一个换行也是这么大，100个也是。删除空格或换行即可。显然删除空格或换行不是最佳的解决办法。

解决办法，用一个父元素包括出现的inline-blocks，然后这是父元素的font-size:0;在给inline-bolcks设置字体大小即可，这种解决方案会多出一个父元素

一个元素设置成inline-block之后，margin:0 auto;就不会再居中了，想要其居中可以给父元素设置text-align:center;

inline-blocks上下结构的margin是不会折叠的，而是叠加的，跟float元素是一样的。

inline-block布局可以是被换行和空格，这可能就是inline-block布局普遍没有被采用的原因吧！

任何元素都可以浮动。浮动元素会生成一个块级框，而不论它本身是何种元素。所以从某种意义上来说，float元素可以理解为带左右定位的inline-block元素。float的包裹性即来源于此
