用法介绍

HTML结构:

  <label for="field_id">Label Text</label><br />
  <input type="text" name="field_id" value=""
	id="field_id">
</p>

CSS结构:
如图所示， 

form .holder-cnt { position: relative;
        margin: 10px 0;}
form .holder-cnt label { position: absolute;
        top: 0; left: 0;
        cursor:	text;}

Javascript调用:

$(document).ready(function(){
  $(".holder-lbl").inFieldLabels();
});

可选项

传递参数有两种实现方式

在调用插件时，设置$.inFieldLabels.defaultOptions.optionName 

$.inFieldLabels.defaultOptions.optionName = "";

在调用时，当成参数传递 

$("label").inFieldLabels({ optionName:value });

fadeOpacity: 值的范围：0.1~1.0
当表单域获取焦点时，label中文本最终的透明度，默认值为 0.5

fadeDuration: 单位为毫秒数
动画的延迟时间，默认值 300

pollDuration: 毫秒数
如果设置的值大于0的话，代码会在指定的时间简称该label相对的表单域的值.

className: 在label上添加的类名，默认值为 false

enabledInputTypes: 被运行的类型的数组
限制input的输入类型. 默认值 [ "text", "search", "tel", "url", "email", "password", "textarea" ]
